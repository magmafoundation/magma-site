# Latest Download

```
GET /api/versions/latest/download
```

Redirects to the file download for the latest version.

## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `type` | string | `installer` | `installer`, `launcher`, or `changelog` |

## Errors

| Status | Description |
|--------|-------------|
| 400 | Invalid type |
| 404 | No versions available or file not found |
