# Download

```
GET /api/versions/{version}/download
```

Redirects to the file download for the specified version.

## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `version` | path | â€” | Version string (e.g. `21.1.33-beta`) |
| `type` | query | `installer` | `installer`, `launcher`, or `changelog` |

## Errors

| Status | Description |
|--------|-------------|
| 400 | Missing version or invalid type |
| 404 | File not found |
