# Direct JAR Download

```
GET /downloads/magma.jar
```

Downloads the latest Magma launcher JAR. Falls back to the regular JAR if the launcher isn't available.

```bash
wget magmafoundation.org/downloads/magma.jar
```

```bash
curl -O magmafoundation.org/downloads/magma.jar
```
