# API Reference

Public endpoints for accessing Magma version data. Useful for launchers, scripts, and integrations.

## Endpoints

- [Versions List](/api-docs/endpoints/versions-list) — List available versions
- [Version Details](/api-docs/endpoints/version-details) — Info about a specific version
- [Download](/api-docs/endpoints/download) — Download a specific version
- [Latest Version](/api-docs/endpoints/latest-version) — Latest version info
- [Latest Download](/api-docs/endpoints/latest-download) — Download the latest version
- [Direct JAR Download](/api-docs/endpoints/direct-jar-download) — Quick URL for latest launcher JAR

## Notes

- Responses are cached for 1 hour.
- Returns `200` with an empty `versions` array when no versions match.
- No strict rate limits currently, but be reasonable.
