# Magma Commands

This page lists the essential Magma commands for server administration.

## Magma-Specific Commands

These commands are unique to Magma and help manage the hybrid server environment:

### /magma

The main Magma command, with these subcommands:

```
/magma version     - Displays Magma version information
/magma mods        - Lists all loaded mods
```
