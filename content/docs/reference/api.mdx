# API

Magma bridges Bukkit/Spigot and NeoForge by mapping Bukkit API calls to Minecraft's codebase while applying NeoForge's hooks. Both mods and plugins run in the same server.

For the website/download API (version info, downloads), see the [Website API docs](/api-docs).
