# Welcome to Magma

Magma is the ultimate Minecraft server software that combines the power of NeoForge mods and Bukkit plugins in one seamless experience. This documentation will help you get started with Magma, configure it to your needs, and troubleshoot common issues.

## What is Magma?

Magma is a hybrid Minecraft server software that allows you to run both NeoForge mods and Bukkit plugins simultaneously. This gives server owners the best of both worlds:

- **NeoForge Mods**: Add new content, items, blocks, and mechanics to your server
- **Bukkit Plugins**: Utilize powerful server management tools, protection systems, and administration plugins

## Key Features

- **NeoForge Mods Support**: Run your favorite NeoForge mods to enhance gameplay and add new content
- **Bukkit Plugins Support**: Utilize powerful Bukkit plugins for server management and administration
- **Optimized Performance**: Enjoy improved server performance with our optimized codebase
- **Easy Setup**: Get your server up and running quickly with simple installation and configuration
- **Active Community**: Join our growing community of server owners and developers for support
- **Comprehensive Documentation**: Access detailed guides to make the most of Magma's features
